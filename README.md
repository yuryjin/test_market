Задача реализована с использованием технологий react, tailwindcss, daisyUI, компонентов из набора shadcn. Удалось реализовать практически всё, в некоторых местах использовал свои решения/упрощения по дизайну

**Запуск проекта осуществляется командами:**

- npm (или pnpm) install
- npm (или pnpm) dev

тесты запускаются по команде **npm (или pnpm) test**
